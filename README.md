# Invitación de Franchesca

Invitación responsive en HTML, CSS y JavaScript, lista para publicar en GitHub Pages.

## Archivos

- `index.html`: estructura y textos.
- `styles.css`: colores, responsive y animaciones.
- `script.js`: calendario, mapa, WhatsApp y cuenta regresiva.
- `assets/logo-franchesca.png`: colocá aquí el logo definitivo. El diseño muestra un logo de texto alternativo si el archivo no existe.
- `assets/referencia-diseno.png`: imagen conceptual original usada como referencia.

## Personalización rápida

En `script.js`, editá el objeto `EVENT`:

- `title`
- `description`
- `location`
- `start` y `end`
- `localDate`
- `mapsQuery`
- `whatsappNumber`
- `whatsappMessage`

Para WhatsApp, escribí el número con código de país y área, sin `+`, espacios ni guiones. Ejemplo argentino: `5491155555555`.

## Publicar en GitHub Pages

1. Creá un repositorio.
2. Subí todos estos archivos respetando las carpetas.
3. En GitHub, entrá en `Settings > Pages`.
4. Elegí `Deploy from a branch`, rama `main` y carpeta `/root`.
5. Guardá y esperá unos minutos.
